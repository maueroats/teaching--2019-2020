    2017-09-14 Indentation

    Indentation is significant in Haskell. Haskell uses indentation to group pieces of code together. This is similar to Python, but there are “false cognates” - similar code that is indented differently in the two languages. In Racket, grouping is done only by parentheses. Indenting to the same level “All grouped expressions must be exactly aligned.” Example 1a: good indentation if x `mod` 3 == 0 && x `mod` 5 == 0 -- good. »
